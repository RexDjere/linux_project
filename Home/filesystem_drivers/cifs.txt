What does CIFS stand for?
Common Internet File System	[[https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/6/html/performance_tuning_guide/s-cifs|[ref 1]]]

What is CIFS and implementation of?
The Server Message Block (SMB) protocol	[[https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/6/html/performance_tuning_guide/s-cifs|[ref 1]]]

What is CIFS used for?
To share file systems, printers, or serial ports over a network	[[https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/6/html/performance_tuning_guide/s-cifs|[ref 1]]]

What sharing does CIFS notably allow?
Notably, CIFS allows sharing files between Linux and Windows platforms regardless of version. [[https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/6/html/performance_tuning_guide/s-cifs|[ref 1]]]
